6. INICIO — HERO SLIDESHOW (index.html, columna derecha)

El hero de index.html muestra en la columna derecha un slideshow automático que cicla por los
5 proyectos del portfolio. Implementado como <a id="heroSlideLink"> con clase hero__img-wrap
y hero__slideshow. Controlado por initHeroSlideshow() en js/main.js.

Comportamiento:
  - Cada slide se muestra 5 segundos (DURATION = 5000 ms) con transición crossfade (opacity 0.8s)
  - Clic en la imagen → navega a projects.html?open=N (abre el modal del proyecto mostrado)
  - El href del enlace se actualiza dinámicamente con cada cambio de slide
  - Pausa al pasar el cursor encima (mouseenter): congela la barra de progreso en su posición exacta
  - Reanuda desde donde se dejó al retirar el cursor (mouseleave), sin reiniciar
  - Puntos de navegación (dots) en la esquina inferior derecha: clic para saltar a cualquier slide
  - Barra de progreso dorada (2px) en la parte inferior: se llena linealmente en 5 s
  - Caption en la parte inferior: muestra categoría (acento dorado) y nombre del proyecto
  - Caption bilingüe: se actualiza automáticamente al cambiar idioma ES/EN
  - En móvil (<900px) el hero visual está oculto (display:none); el slideshow no afecta

Para añadir imágenes a un proyecto:
1. Colocar los archivos PNG/JPG en su carpeta correspondiente (assets/img/<proyecto>/)
2. Los vídeos van en assets/videos/<proyecto>/ o como enlaces YouTube embed
3. Añadir data-img="ruta" y data-gallery="ruta1,ruta2,..." al div .project-item
4. Para YouTube: usar URL https://www.youtube.com/embed/VIDEO_ID (con |vol:0.X opcional)
5. Para miniatura con fondo desenfocado: añadir div.hero__slide-blur + img con object-fit:contain + position:relative

16. NOTAS TÉCNICAS

Google Fonts cargadas: Bebas Neue, Cormorant Garamond (300, 400, italic), Raleway (300, 400, 500)
Variables CSS principales: --accent, --accent-dim, --bg, --bg-card, --text, --muted
Acento por página: bloque style en head de cada HTML sobreescribe --accent
Layout compartido: js/layout.js inyecta nav y footer en todos los HTML
Traducciones: objeto TRANSLATIONS en js/main.js, atributo data-i18n en HTML
Auto-apertura de modal: projects.html?open=N → abre proyecto índice N; URL se limpia con history.replaceState
Galería de imágenes: atributo data-gallery con rutas separadas por coma
  - Vídeos HTML5 (.mp4/.webm): se renderizan como <video autoplay muted loop playsinline>
  - Vídeos YouTube: URL https://www.youtube.com/embed/ID → se renderizan como <iframe>
  - Control de volumen: sufijo |vol:0.85 en la ruta (YouTube via postMessage, HTML5 via .volume)
Miniatura con fondo desenfocado: div.hero__slide-blur (position:absolute, filter:blur(18px) brightness(0.35))
  + img con object-fit:contain y position:relative (position:relative es obligatorio para que el img
  quede por encima del blur en el orden de capas CSS; sin él el blur tapa la imagen)
Hero slideshow: función initHeroSlideshow() en js/main.js; window._heroSlideshowUpdateLang() para
  actualizar el caption al cambiar idioma; usa setTimeout (no setInterval) para poder reanudar
  desde el tiempo exacto tras un hover
About page: columna foto 0.6fr · aspect-ratio 3/3.5 · textos 1.1rem / 1.18rem · timeline ampliado
Hosting recomendado: Netlify Drop (gratis) + dominio .com o .art (~10€/año)
